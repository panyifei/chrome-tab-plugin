chrome的标签页插件，用自己编写的才过瘾嘛~

## 整个写的过程研究参见[我的学习分享](https://github.com/panyifei/learning/blob/master/JS%E4%BB%A3%E7%A0%81%E5%BA%93/Chrome%E6%8F%92%E4%BB%B6%E7%BC%96%E5%86%99.md)

## 安装使用
将项目clone到本地

然后访问chrome://extension

然后打开开发者选项

然后安装，选择好根目录就行了
